### 前言

TypeScript(以下简称 TS)是一种由微软开发的自由和开源的编程语言。它是 JS 的一个超集。 C#的首席架构师 Anders Hejlsberg，已工作于 TS 的开发。
TS 扩展了 JS 的语法，所以任何现有的 JS 程序可以不加改变的在 TS 下工作。TS 是为大型应用之开发而设计，而编译时它产生 JS 以确保兼容性。
Google 公司与微软合作，用 TS 写了 Angular2
